## 🚀 Getting Started

### Prerequisites
- Python 3.8+ (for Python version)
- Java 8+ (for Java version)
- API keys (if required by weather service)

### Installation

**Python:**
```bash
# Clone and navigate to project
cd weather-forecast
python weather.py
```

**Java:**
```bash
# Compile
javac WeatherApp.java
# Run
java WeatherApp
```
